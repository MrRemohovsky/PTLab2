# Лабораторная 2 по дисциплине "Технологии программирования"

## Постановка задачи

Разработка веб-приложения интернет-магазина детских игрушек с реализацией системы промокодов. Покупатель может использовать промокоды для получения скидок на товары размером 5%, 10% или 15% в зависимости от типа промокода.

## Основная функциональность:

    Каталог товаров с описанием и ценами

    Система применения промокодов

    Автоматический расчет скидок (5%, 10%, 15%)

    Вывод финальной цены со скидкой

## Используемые технологии

    Язык программирования: Python 3.8

    Фреймворк: Django

    База данных: PostgreSQL

    Тестирование: Django Test Framework (django.test)

    Система контроля версий: Git/GitHub

## Реализованные расширения

Модель промокодов:

Добавлена модель PromoCode, которая позволяет управлять различными типами промокодов. Каждый промокод имеет уникальное имя и связанный процент скидки (5%, 10% или 15%). При применении промокода система автоматически пересчитывает стоимость товаров.
